# Case Management Dashboard

## 📌 Project Description
This is a Case Management Dashboard built using Angular.  
It allows users to manage cases with full CRUD functionality.

## 🚀 Features
- Add new case
- Edit existing case
- Delete case
- Search/filter cases
- Data stored using localStorage

## 🛠️ Technologies Used
- Angular
- TypeScript
- HTML
- CSS

## ▶️ How to Run
1. Install dependencies:
   npm install

2. Run the project:
   ng serve

3. Open in browser:
   http://localhost:4200

## 🎯 Purpose
This project was built to practice Angular concepts like:
- Data Binding
- Components
- Directives
- Forms handling
